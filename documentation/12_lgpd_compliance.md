# LGPD

## Dados pessoais tratados
Nome, e-mail, CPF, telefone, data de nascimento, métricas físicas.

## Recomendações
- Base legal explícita por finalidade
- Política de retenção/exclusão
- Registro de consentimento para notificações
- Minimização de exposição de dados sensíveis
- Logs de auditoria para ações administrativas
