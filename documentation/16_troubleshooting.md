# Troubleshooting

- Erro `Academia não encontrada`: validar slug/schema/token
- `Token inválido`: conferir expiração e `secret_key`
- SMTP não envia: revisar credenciais e porta TLS
- Falha nos testes: conferir dependências (`pip install -r requirements.txt`)
- Inadimplência inconsistente: rodar atualização de matrículas e checar vencimentos
